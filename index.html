How to play

Game starts by pressing Start button. Players are taking turns in dice rolling. Each player can roll unlimited times using roll button and rolled number keeps adding to the current player's score. But if a player rolls 1, he loses all current score and other player takes turn. To save the current score player needs to press hold button and then player's current score is added to his total score and the other player takes turn.
