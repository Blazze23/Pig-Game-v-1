Pig game is a game where players are rolling a dice. Player can roll unlimited number of times and every dice roll keeps adding to his current score. If number 1 is rolled then current score is lost and other player takes turn. Player can save his current score and then other player takes turn. First player to reach 50 points wins. Game duration can be changed in settings.
